## Important points to remember (Java Tip)
- 입력, 출력, 기능을 알 수 있도록 메소드에 주석 기입
- 변수가 null인지 확인해야함
- 리터럴과 비교하는 함수를 쓸때는 리터럴을 왼쪽에 써야함 (NullPointException 방지를 위해서)
- if문 이후에 코드가 1줄만 있어도 코드블럭을 만드는게 good
